# 🌍 AtmoPulse — Real-Time AQI Monitor

A premium, real-time Air Quality Index (AQI) monitor for cities worldwide. Built with a modern dark UI, interactive Leaflet map, and powered by **WAQI** and **OpenAQ** APIs.

**➡ [Live Demo](https://atmopulse.web.app)**

---

## ✨ Features

- 🗺️ Interactive dark map with glow-effect AQI markers
- 🔍 City search with live autocomplete suggestions
- 📊 Detailed pollutant panel — PM2.5, PM10, O₃, NO₂, CO, SO₂ and more
- 🌤️ Live weather data — Temperature, Humidity, Wind, Pressure
- 📈 Session AQI trend chart with sparkline visualization
- 🔔 Custom AQI alerts with push notification support
- 🕘 Search history with quick re-search
- 🔑 API key management (requires sign-in)
- 👤 Google Sign-In + Email/Password authentication
- 🏭 Multi-source data — WAQI + OpenAQ combined
- 📱 Fully responsive — works on mobile and desktop
- ⚡ Real-time refresh with cooldown timer
